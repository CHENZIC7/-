# 校园微农园智能监测系统

基于AIoT的校园微农园智能监测系统设计与实现 - 物联网通讯技术期末作业

## 项目简介

本项目是一个网页端的校园微农园智能监测系统，模拟展示了物联网技术在农业中的应用。系统包含以下功能：

- 📊 实时数据仪表盘 - 展示温度、湿度、土壤湿度、光照、pH、CO₂等传感器数据
- 📈 历史趋势图表 - 使用Chart.js展示24小时数据变化趋势
- 🔧 设备管理 - 查看各传感器节点状态和网关信息
- ⚠️ 预警中心 - 显示系统预警和操作记录
- 🎮 控制面板 - 远程控制灌溉、遮阳、通风、补光等设备
- 🤖 AI智能模式 - 根据传感器数据自动决策控制设备

## 技术栈

- HTML5
- CSS3 (响应式设计)
- JavaScript (原生)
- Chart.js (图表库)

## 快速开始

### 方法一：使用批处理文件（推荐）

1. 双击运行 `启动服务器.bat`
2. 在浏览器中访问 http://localhost:8000

### 方法二：手动启动

1. 打开命令行/终端
2. 进入项目目录：
   ```bash
   cd 物联网通讯技术期末作业/src
   ```
3. 启动Python HTTP服务器：
   ```bash
   python -m http.server 8000
   ```
4. 在浏览器中访问 http://localhost:8000

### 方法三：直接打开HTML文件

直接双击 `src/index.html` 文件即可在浏览器中打开（部分功能可能需要服务器环境）。

## 功能说明

### 1. 数据仪表盘
- 实时显示6种传感器数据
- 自动每3秒更新数据
- 颜色指示数据是否在适宜范围内

### 2. 历史趋势
- 温湿度24小时变化曲线
- 土壤湿度与光照变化对比
- CO₂浓度柱状图
- pH值变化趋势

### 3. 设备管理
- 显示5个传感器节点状态
- 网关设备运行状态
- 通信方式、信号强度、电池电量等信息

### 4. 预警中心
- 土壤湿度预警
- 电量低提醒
- 温度过高预警
- 操作记录查看

### 5. 控制面板
- 灌溉系统控制
- 遮阳系统控制
- 通风系统控制
- 补光系统控制
- AI智能模式开关
- 远程通知设置

### 6. AI智能模式
- 自动监测传感器数据
- 根据阈值自动控制设备
- 智能决策提示

## 模拟数据说明

本系统使用模拟数据展示功能，数据会在合理范围内随机波动：
- 温度：18-32°C
- 湿度：40-80%
- 土壤湿度：30-90%
- 光照：500-6000 lux
- pH值：5.5-7.5
- CO₂：400-1200 ppm
